tsadj_SetValueAdjustmentPanel

void tsadj_SetValueAdjustmentPanel (CAObjHandle sequenceContext, long resourceID, double value, short *errorOccurred, long *errorCode, char errorMessage[]);

Parameters

Input
Name Type Description
sequenceContext CAObjHandle For use with TestStand: The TestStand sequence context ("ThisContext")
For use in other environments: Must be zero.
resourceID long Resource Identification
value double
Output
Name Type Description
errorOccurred short (passed by reference) Error Flag.
Return :
0 -> No Error
1 -> Error Occured
errorCode long (passed by reference) Error code. Negative values indicate an error.
Return:
0 -> No Error
< 0 -> Error
errorMessage char [] Error message buffer (1024 bytes)
In case of an error, the error message is copied into the error message buffer. The buffer remains unchanged if the function returns successfully.
